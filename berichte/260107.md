# Replit und Git
1. In repl das Tool git öffnen.
2. in Github ein neues Repositorie erstellen
3. in Github bei dem Repositorie in settings unter Pages bei Branch main auswählen und save anklicken
4. Dann den Link von dem Github Repositorie in das Url Feld bei Git in Repl einfügen
5. Bei dem Commit feld einen Commit eintragen
6. Danach auf Das blaue Feld "Stage and commit all changes" klicken
7. dann oben auf sync changes klickt sind  die Daten  wenn man die Seite von Github neu lädt auch in dem Github repositorie.
